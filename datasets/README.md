# Indeterminacy Research Datasets

Datasets used in Validating LLM-as-a-Judge Systems under Rating Indeterminacy

## Datasets
- **Civil Comments**: Toxicity detection
- **ChaosNLI**: Natural language inference (MNLI, SNLI, AlphaNLI)
- **SummEval**: Summarization quality (relevance, coherence, consistency, fluency)
- **QAGS**: Factuality assessment
- **TopicalChat**: Dialogue evaluation
